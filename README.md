# 🚀 CreativeHub - Modern Agency Landing Page

A modern and responsive agency landing page built using **HTML5** and **Tailwind CSS CDN**. This project showcases a professional business website with a clean user interface, pricing plans, feature highlights, contact form, and responsive design.

---

## Live Demo - https://wasim2934.github.io/CreativeHub-Modern-Agency-Landing-Page


## 📌 Project Overview

CreativeHub is a modern landing page designed for digital agencies, startups, freelancers, and businesses looking to establish an online presence. The website focuses on clean layouts, responsive design, and user-friendly navigation.

## ✨ Features

* Responsive Navigation Bar
* Full-Screen Hero Section
* Smooth Scrolling Navigation
* Feature Cards Section
* Pricing Plans Section
* Contact Form UI
* Professional Footer
* Mobile-Friendly Design
* Modern Gradient Effects
* Hover Animations & Transitions
